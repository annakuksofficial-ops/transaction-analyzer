# Transaction Analyzer

Программа для анализа банковских транзакций из Excel-файла.

## Что умеет программа

### Главная страница
- Пишет приветствие (Доброе утро, Добрый день, Добрый вечер, Доброй ночи)
- Показывает по каждой карте: последние 4 цифры, сколько потратили, кешбэк
- Показывает топ-5 самых больших трат
- Показывает курсы валют (USD, EUR)
- Показывает цены акций (AAPL, AMZN, GOOGL, MSFT, TSLA)

### Страница События
- Показывает общую сумму расходов
- Показывает траты по категориям (топ-7, остальное в "Остальное")
- Показывает переводы и наличные
- Показывает общую сумму поступлений
- Показывает поступления по категориям

### Сервисы
- **Простой поиск** - ищет транзакции по слову в описании или категории
- **Выгодные категории кешбэка** - считает, сколько кешбэка можно получить по каждой категории за месяц
- **Поиск по телефонным номерам** - находит транзакции, где в описании есть номер телефона

### Отчеты
- **Траты по категории** - показывает траты по выбранной категории за последние 3 месяца
- **Траты по дням недели** - показывает средние траты по дням недели за последние 3 месяца

## Как установить

1. Скачай проект
2. Открой терминал в папке с проектом
3. Установи зависимости:
pip install pandas requests python-dotenv openpyxl pytest pytest-mock pytest-cov
4. Положи файл `operations.xlsx` в папку `data`

## Как запустить

В терминале выполни:
python -m src.main

## Как запустить тесты

pytest tests/ -v

## Результаты работы

При запуске программа покажет:
- Приветствие
- Сколько карт найдено
- Курсы валют
- Поиск по слову
- Отчет по категории
